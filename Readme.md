Samvaad is a real-time chat application built using HTML,CSS,Javascript, and modern web technologies. It allows users to authenticate securely and communicate instantly through a clean and responsive user interface.

### Features

1. User can signup and login using username/email and password.
2. User can create and Delete Room. However Delete room is only available to that user who created the room.
3. All users can join and leave the room.
4. Real Time Messaging done using socket.io.
